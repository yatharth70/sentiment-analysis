# Sentiment Analysis & Visualization Tool 💬

## 📌 Problem
Understanding customer sentiment from text data (reviews, feedback, social media) is crucial for improving products, services, and user experience.

## 💡 Solution
Built a sentiment analysis tool using NLP techniques to classify text into Positive, Negative, or Neutral categories and visualize overall sentiment trends.

## 🛠️ Tech Stack
- Python
- NLTK (VADER Sentiment Analyzer)
- Pandas, NumPy
- Matplotlib

## ⚙️ Approach
- Preprocessed text data (cleaning, tokenization)
- Used VADER Sentiment Intensity Analyzer for scoring
- Classified sentiments based on polarity scores
- Performed word-level sentiment analysis
- Visualized sentiment distribution using graphs

## 📊 Results
- Accurately classified text into sentiment categories  
- Generated insights from textual data  
- Visualized sentiment distribution for better understanding
